# AkariNet-PocketTTS


An **AkariNet-compatible speech synthesis module** written in JavaScript, for local text-to-speech generation using ONNX Runtime Web.

## Credits
This project is a derivative work based on the [pocket-tts-web Hugging Face Space](https://huggingface.co/spaces/KevinAHM/pocket-tts-web) developed by **KevinAHM**. The core logic for this module was adapted from his original demo.
